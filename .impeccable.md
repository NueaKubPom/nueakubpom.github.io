## Design Context

### Users
ผู้ใช้งานหลักคือคนที่เข้ามาผ่านลิงก์รวมของสตูดิโอ เพื่อดูช่องทางติดต่อ ผลงาน และเครื่องมือเด่นได้อย่างรวดเร็วในหน้าเดียว

### Brand Personality
Creative, modern, polished. น้ำเสียงควรดูมั่นใจ ทันสมัย และมีความเป็นสตูดิโอดิจิทัลที่ใส่ใจรายละเอียด

### Aesthetic Direction
Dark mode เป็นหลัก โดยใช้แนว glass UI ที่ดูสะอาด คม และมี motion นุ่มลื่นแบบร่วมสมัย ผสม subtle glow, layered blur, และ interactive hover ที่ช่วยเพิ่มความรู้สึกพรีเมียมโดยไม่รก

### Design Principles
1. ให้แบรนด์เด่นตั้งแต่ first viewport
2. ทุก action ต้องมองเห็นและกดได้เร็ว
3. ใช้ glass, glow, และ motion อย่างมีจังหวะ ไม่ฟุ้งเกินจำเป็น
4. รักษา visual hierarchy ให้ชัดทั้ง desktop และ mobile
5. เพิ่มความทันสมัยโดยไม่ทำให้การใช้งานซับซ้อน
